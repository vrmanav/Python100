- This repository is a collection of projects and exercises based on [Dr. Angela Yu's](https://www.udemy.com/user/4b4368a3-b5c8-4529-aa65-2056ec31f37e/) "100 Days of Code - The Complete Python Pro Bootcamp" course on [Udemy](https://www.udemy.com/course/100-days-of-code/).
- You'll find a detailed breakdown of my daily coding progress, challenges faced, and lessons learned. It's a journey log, so you can see how I evolve over the 100 days.
- The course is project-based, and I'll be sharing the code for each project as I complete them. Feel free to explore, provide feedback, or use them as a reference for your own projects.

- You will find directories organized by day, each containing the code and projects completed on that specific day. Additionally, there may be supplementary resources and notes to enhance your learning experience.

## Requirements

To get started with this repository and follow along with the course, you'll need to have the following software and libraries installed on your machine:

Make sure you have [Python](https://www.python.org/downloads/) & [Pip](https://pip.pypa.io/en/stable/installation/) (as package manager) installed.

### Libraries

Use the following command to install the necessary Python libraries:

```bash
  PrettyTable: pip install prettytable

  Pandas: pip install pandas

  Pyarrow: pip install pyarrow

  OpenPyXl: pip install openpyxl
```
